#  Beyond the Mean: Fisher-Orthogonal Projection for Natural Gradient Descent in Large Batch Training

Official code for  an Oral presentation at AAAI 2026
**"Beyond the Mean: Fisher-Orthogonal Projection for Natural Gradient Descent in Large Batch Training"**  
by Yishun Lu, Wes Armour

📄 Full Version: [arXiv:2508.13898](https://arxiv.org/abs/2508.13898)

---

## Usage

Run the main experiment script:
```
./run_paper.sh
```

This script reproduces the paper experiments with the default settings. You may need to adjust hyperparameters for different tasks.

Acknowledgement: The CIFAR pipeline uses [Cutout](https://github.com/uoguelph-mlrg/Cutout).

Dependencies are listed in `requirements.txt`.
